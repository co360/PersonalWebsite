Android e-Commerce Shopping Application

Description

An Android Application that allows users to shop for products in different categories, users can register an account and login. The data will get saved to a Firebase Database and then the users can login with the credentials from Firebase.

Experience

This project was my first ever big Android application that I have developed as part of a module called Mobile Application Development at University. I got the chance to use many interesting technologies and features to implement a coherent and well structured application.

Maintenance

Updates and bug fixes will be made in the near future, also new features will be added to improve the funcionality of the application as well as refining the User Interface a bit.
